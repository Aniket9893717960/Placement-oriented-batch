                        WHAT IS SPRING FRAMEWORK?
Spring is a lightweight and popular open source Java based framework developed by Rod johnson
in 2003. It is used to develop enterprise level applications. It provides support to many 
other frameworks such as Hibernate, Tapestry, EJB, JSF, Struts, etc.
It is also called a framework of frameworks. It's an Application framework and IOC
(Inversion of Control) container for the java plateform.
It's spring containes several modules like IOC, AOP, DAO, CONTEXT, WEB MVC, etc.
